# LT4532B‑351MGN / LT4532B‑351MGF 作为 CH318 隔离变压器的选型说明

本说明文档用于记录在 IsolaPurr USB Hub 项目中，为 **CH318T 高速 USB 隔离延长芯片** 选择隔离变压器的依据与结论。相关数据请参考：

- `docs/datasheets/ch318t-datasheet.md`  
- `docs/datasheets/lt4532b-351mgn-datasheet.md`  
- `docs/datasheets/lt4532b-351mgf-datasheet.md`

## 1. 应用背景与需求

- CH318T 用作高速 USB 信号隔离/延长控制芯片，两颗 CH318T 之间通过 **一只网络变压器** 或耦合电容实现差分信号的隔离与传输。  
- 本项目采用「单变压器、两侧各一颗 CH318T」的方案，只需要完成：  
  - 差分信号的 **AC 耦合与隔离**；  
  - 适配 CH318 推荐的网络变压器应用电路；  
  - 提供足够的带宽与合适的漏感 / 寄生参数，保证 USB 2.0 高速链路稳定工作。  
- 该链路 **不传输 PoE 供电，不叠加大直流偏置电流**，仅为差分小信号传输使用。  

因此，隔离变压器的核心关注点为：

- 匝比 1:1、中心抽头适配 CH318 参考设计；  
- 足够的 OCL（350 µH 级别）和带宽，满足高速差分信号；  
- 较小的漏感与绕组间电容（Cww），减小失真与共模串扰；  
- 具备 ≥1.5 kVrms 的绝缘耐压；  
- SMD 封装便于版图布线，与现有布局尺寸匹配。

## 2. LT4532B‑351MGN / MGF 关键参数概述

根据两份原厂数据手册（Markdown 版本）：

- 型号：**LT4532B‑351MGN T6 / LT4532B‑351MGF T6**  
- 匝比：**1CT : 1CT**  
- OCL：**350 µH Min @ 100 kHz, 0.1 V**，在各自规定的 DC 偏置电流下保持 350 µH：  
  - MGN：**8 mA DC bias**  
  - MGF：**10.5 mA DC bias**  
- 绕组间电容 Cww：**45 pF Max**  
- DCR：**4.5 Ω Max**  
- 插入损耗：  
  - ≤ −1.0 dB @ 1–100 MHz  
  - ≤ −1.2 dB @ 100–125 MHz  
- 回波损耗（Return Loss）：  
  - ≥ −18 dB @ 1–40 MHz  
  - ≥ −17 dB + 10·log(F/40) @ 40–100 MHz  
- 绝缘耐压：**1500 Vrms**  
- 封装：SMD‑6P，尺寸 4.6 mm × 3.4 mm，三路绕组引脚引出，完全一致。

两者在小信号 AC 参数上完全相同，封装与引脚兼容。

## 3. 标准与应用差异：IEEE 802.3ab vs 802.3af

- **LT4532B‑351MGN T6**  
  - 数据手册标注：**Meets IEEE 802.3ab Standards**  
  - 即面向 **1000BASE‑T 千兆以太网** 应用，不强调 PoE 供电能力；  
  - OCL 保证条件是 **8 mA DC bias**。

- **LT4532B‑351MGF T6**  
  - 数据手册标注：**Meets IEEE 802.3af Standards**  
  - 明确给出 **350 mA Current Capability**，典型为 **PoE（Power over Ethernet） 场景**准备；  
  - 在 **10.5 mA DC bias** 下仍保证 350 µH OCL。  

结合可知：

- MGF 型号在 **有直流偏置 / PoE 电流** 的情况下，对磁芯饱和和 OCL 下降的裕量更大；  
- 在 **无 PoE、近零 DC 偏置** 的纯信号传输场景下，两者的差异主要体现在“额定电流能力标称”而非小信号参数。

CH318 的 CH318T 应用（见 `ch318t-datasheet.md` 第 8 章）中，网络变压器只是用于高速 USB 信号的隔离延长，不承担供电功能，因此工作点属于 **小信号 AC、近零直流偏置**，与 MGN/MGF 两个版本的差异不敏感。

## 4. 与 CH318 应用的匹配性分析

CH318 官方应用中，网络变压器需要满足：

- 1:1 差分耦合；  
- 对应 USB 差分对的阻抗环境（板上布线 + 变压器寄生）在高速下保持可接受的眼图与抖动；  
- 具有足够的隔离耐压满足系统安规需求。

LT4532B‑351MGN / MGF 的特性与之对比：

- 匝比 1CT:1CT，且中心抽头拓扑与以太网 LAN 变压器一致，适配 CH318 推荐的变压器接法。  
- OCL=350 µH Min（小信号条件下更高），对于差分信号在几十 MHz 带宽内的耦合足够，同时避免过度低频衰减。  
- 插入损耗和回波损耗指标在 1–100 MHz 范围内优于或接近常规千兆 LAN 磁件，明显好于 CH318 这类 USB‑over‑twisted‑pair 场景的要求。  
- Cww=45 pF Max，对 USB 差分线来说不会带来不可接受的共模耦合或带宽限制。  
- 1500 Vrms 耐压满足项目中对信号隔离的基本要求（与 CH318 芯片本身的绝缘协同考虑）。

由于 CH318 差分链路中 **没有 PoE 电流，也不在变压器上叠加大直流偏置**，MGN 型号的 DC 偏置条件（8 mA）已经有足够余量；MGF 型号在本应用中不会发挥其 PoE 电流能力优势，但可作为电气等效的更高规格替代。

## 5. MGN vs MGF 的选型策略

结合本项目当前与潜在需求，可以给出以下策略：

1. **基础选型（BOM 主型号）**  
   - 选用 **LT4532B‑351MGN T6** 作为默认器件：  
     - 满足 CH318T 隔离延长应用所需的全部小信号、电气与绝缘指标；  
     - 无 PoE 需求时，不需要为额外的电流能力付出潜在成本溢价。

2. **替代料策略（Drop‑in 替代）**  
   - 如供应链上 MGN 出现缺货、交期过长等问题，可用 **LT4532B‑351MGF T6** 直接替代：  
     - MGF 与 MGN 封装、引脚、版图完全兼容；  
     - 小信号参数在 datasheet 中完全一致；  
     - 在 CH318 应用场景下，两者对电路行为的影响可以认为等效。  
   - 原理图与 PCB 封装统一标注为「**LT4532B‑351MGN T6（ALT: LT4532B‑351MGF T6）**」即可。

3. **PoE 场景的扩展可能**  
   - 本项目当前不使用 PoE，但若未来考虑在同一 PCB 上复用该变压器进行 PoE 实验或扩展（例如共用磁件、留出中心抽头接入 PSE/PD 电路），则优先使用 **LT4532B‑351MGF** 会更安全。  
   - 现阶段设计中，CH318 差分链路与任何 PoE 功能是解耦的，因此文档仅记录这一潜在方向，而不作为当前版本的设计前提。

## 6. 结论

- **LT4532B‑351MGN T6 适合作为 CH318T 差分隔离链路的主选网络变压器**，在带宽、OCL、插入损耗、回波损耗及绝缘耐压等方面均满足或优于需求。  
- **LT4532B‑351MGF T6 在本应用中可以视为电气等效的高规格替代件**，兼容同一原理图与 PCB 封装，可作为 BOM 中的第二来源。  
- 由于 CH318 应用中无 PoE 需求、无大直流偏置电流，两者在实际电路表现上的差异可以忽略不计，选型主要由成本与供货情况驱动。

后续如项目在信号完整性验证中发现对变压器参数有更高要求（例如对 Cww、漏感或高频损耗异常敏感），可以在本文件基础上补充实际测量结果与仿真数据，但无需更换器件系列。
