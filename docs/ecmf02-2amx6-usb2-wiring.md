# ECMF02-2AMX6（ST）USB2.0 D+/D- 引脚互换与布线要点

结论：**可以把 `1&6` 这一路当作 D-，把 `2&5` 这一路当作 D+（或反过来）**；器件内部两路是对称的，哪一路承载 D+ / D- 并不影响功能。关键是**同一根信号线必须始终走在同一对引脚上**，不要交叉。

## 1. 引脚对应（Top view，按 ST datasheet Figure 1）

> ST ECMF02-2AMX6 datasheet：Doc ID 17815 Rev 2（June 2011）

```
Top view (• = pin 1)

   _______
1 |•    | 6
2 |     | 5
3 |_____| 4

1 = D+     6 = D+
2 = D-     5 = D-
3 = GND    4 = NC
```

工程上可把两路理解为两条“直通通道”：

- 通道 A：`1 ↔ 6`
- 通道 B：`2 ↔ 5`

## 2. 你遇到的“互换”到底指什么

你说的做法（为了更好布线）：

- `1&6` 全程接 **D-**
- `2&5` 全程接 **D+**

这是**把 D+/D- 分配到另一条通道上**，D+ 仍是 D+、D- 仍是 D-（只是走了不同的通道），**是 OK 的**。

不 OK 的情况是“交叉穿越”，例如：

- D+ 进 `1` 却从 `5` 出；或 D- 进 `2` 却从 `6` 出

这会在器件内部把差分对 **对调/反相**，USB2.0 通常会直接不工作或不稳定。

## 3. 布线建议（USB2.0）

- 器件尽量靠近 **接口/外部电缆** 一侧放置（优先保护接口）。
- D+/D- 做差分对等长、等阻抗、同层走线；避免不必要的过孔与长支路（stub）。
- `GND`（pin 3）用**短而宽**的回流路径接地平面，建议就近打多颗过孔降低电感。
- `NC`（pin 4）按 datasheet：**保持悬空不连接**。
